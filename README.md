# CSCI 331 - Web Development Final Project

### Overview:

This application was created to help users find doable recipes based on available ingredients on-hand. This is the final project for CSCI 331 - Web Development.

### Technology Stack:

- Host: Linux
- Frontend/backend framework: Svelte w/ Typescript
  - TailwindCSS for styling
- Database: SQLite
- Containerization: Docker
- APIs: [TheMealDB](themealdb.com)
